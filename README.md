# Tencent-DataScience-Projects 腾讯算法-数据科学岗位专属项目
本仓库为投递腾讯算法-数据科学岗位专属，包含本人简历中3个核心数据科学与量化分析项目的完整代码、文档与报告，适配算法数据科学、金融科技岗位需求。

## 项目1：基于机器学习算法对二手房价预测项目
📂 项目文件夹：House-Price-Prediction
- 技术栈：Python 3.9.13、Jupyter Notebook、Pandas、Scikit-learn、Matplotlib/Seaborn
- 项目背景：针对北京二手房房价人工预测误差大、缺乏数据化支撑的问题，搭建机器学习房价预测模型，为房产市场价格评估提供精准算法方案
- 核心成果：处理23677条原始数据，完成数据预处理与特征工程；对比多模型评估，随机森林模型R²达0.8639，挖掘房价核心影响特征
- 附件说明：包含完整项目代码、项目PPT

## 项目2：中美股市对比分析项目
📂 项目文件夹：US-China-Stock-Analysis
- 技术栈：Python 3.9.13、Jupyter Notebook、Pandas、Scipy、Matplotlib
- 项目背景：针对中美资本市场收益波动、市场有效性差异不明确的问题，开展跨市场量化对比研究，为全球金融市场分析提供实证参考
- 核心成果：选取中美核心指数与科技龙头股数据，完成多维度金融数据清洗与规整；通过假设检验、自相关检验、游程检验等量化分析，量化验证中美市场效率与量价特征
- 附件说明：包含完整项目代码、分析报告PDF、图表文件等相关数据文件

## 项目3：中国市场上市公司业绩与股价关系研究
📂 项目文件夹：A-Stock-Quant-Analysis
- 技术栈：Python 3.9.13、Jupyter Notebook、Pandas、Matplotlib
- 项目背景：针对A股市场上市公司业绩与股价关联规律不明确的问题，开展多行业龙头企业业绩与股价联动研究，为资本市场分析提供量化参考
- 核心成果：选取六大行业10家A股龙头企业2020-2024年数据，完成Pearson相关性分析，量化挖掘行业差异化关联特征，实现分析流程自动化
- 附件说明：包含完整项目代码、分析报告PDF、图表文件等相关数据文件
